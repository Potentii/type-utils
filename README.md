# Type Utils

[![NPM Version][npm-image]][npm-url]

> Type utilities

<br>

## Installing
```
$ npm install @potentii/type-utils
```

---

## Usage

```javascript
import { TypeOf } from '@potentii/type-utils';

TypeOf.checkValue('my-value', 45, 'number', true);
```
